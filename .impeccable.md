## Design Context

### Users
Club organizer managing dinner club event scheduling. Admin-only dashboard — members only see the voting page. Organizer is checking results, confirming slots, and managing polls. Practical tool, not a consumer product.

### Brand Personality
Efficient, clear, utilitarian

### Aesthetic Direction
Clean admin tool. No brand identity needed — prioritize information density, scanability, and fast workflows over personality. Light mode default. Should feel like a well-made internal tool (Linear, Notion admin) not a marketing site.

### Design Principles
1. **Information first** — data should be immediately scannable, not buried in chrome
2. **Fast workflows** — minimize clicks, maximize keyboard efficiency
3. **Quiet confidence** — well-spaced, well-typed, no decorative noise
4. **Clear hierarchy** — the most important number/action should dominate each view
